# Đặt Sân Online - Frontend

Đây là phần Frontend của dự án Hệ Thống Đặt Sân Thể Thao Online, được xây dựng bằng React 19 và Vite.

Vui lòng xem [README chính của dự án](../README.md) để biết chi tiết về công nghệ, cấu hình và hướng dẫn cài đặt đầy đủ.

## Lệnh Chạy Nhanh

- `npm install`: Cài đặt dependencies.
- `npm run dev`: Chạy môi trường phát triển (mặc định tại `http://localhost:5173`).
- `npm run build`: Build dự án cho production.
